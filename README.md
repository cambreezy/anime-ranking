# Anime Rankings

A tournament-style anime ranking app. Pick your anime, vote in head-to-head matchups, and see your personal rankings with history and trend charts.

## Features

- Head-to-head voting to rank your anime list
- Leaderboard with win counts
- Head-to-head stats between any two anime
- Snapshot history with a bump chart to track ranking changes over time
- Export/import rankings as JSON
- Fully offline — no backend, all data in browser localStorage

## Deploy (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your app will be live at `https://<username>.github.io/<repo-name>/`

## Run Locally

```bash
python3 serve.py
```

Then open [http://localhost:3456](http://localhost:3456).
